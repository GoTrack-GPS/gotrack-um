# Báo cáo nhiệt độ
> Chỉ có những xe lắp cảm biến nhiệt độ mới có dữ liệu.

Nếu người dùng muốn xe báo cáo nhiệt độ :  
* Vào **Báo cáo >  Báo cáo > Nhiệt độ**.
Hiện ra màn hình **Báo cáo nhiệt độ** như hình dưới đây :
